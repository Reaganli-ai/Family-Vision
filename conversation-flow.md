# 家庭愿景工坊 — 对话 + 交互规格文档

> **定位**：本文档是 Word 业务母稿（《快照手册 & 定位罗盘 v2》+ 《工作纸》）的数字化交互实现稿。
> 所有方法论、模块顺序、核心问题和最终产出均以 Word 为准，本文档不发明新方法论。
>
> **核心设计原则**：
> 1. 所有选项/标签/候选由 AI 根据用户上下文动态生成，**不硬编码**
> 2. AI 不替用户做价值选择，AI 负责理解、追问、提炼、给 hint
> 3. 命名权在用户，AI 给参考方向帮助打开思路
> 4. W 模块像家庭战略咨询师多轮深挖，不是一张卡片收完就走

### Word 快照模板（原文）

**家底快照**：我们是一个在**经济**上偏向 `___`，在**文化**上充满 `___`，在**社会**上连接着 `___` 的家庭。我们决心**优先**关注 `___` 资本。

**眼光快照**：我们看到的关键未来趋势是「`___`」。我们初步关注的未来素养是「`___`」，它隐约呼应了我们内心对「`___`」的看重。

**根基快照**：我们家族的底层代码，可能是一种「`___`」的生存哲学。它源于对「`___`」的看重。我们决心，在继承其「`___`」内核的同时，勇敢地将其表现形式升级为「`___`」，以托举孩子的未来。

**共识快照**：我们从纷繁的价值观中，锚定了「`___`」、「`___`」与「`___`」作为家庭教育的三大战略基石。我们的战略罗盘指向【 】内核 /【 】创造 /【 】连接。

**教育愿景**（罗盘最终产出）：我们，【姓名代码】家庭，将运用【家庭优势资本 S】，秉承【升级后的家族精神 W】，在充满【未来关键趋势 N】的时代，坚定选择【价值观方向 E】的道路，全力将孩子培养成一个【结合核心素养与价值基石的、鲜活的人像描述】的人。

#### N-02 · AI 趋势转译 → 用户排序

| 字段 | 内容 |
|------|------|
| **对应 Word** | 工作纸第三步"趋势转译"——从信号推断长期趋势 |
| **目标** | AI 基于用户的信号和情绪生成个性化趋势解读 → 用户排序确认 |
| **任务类型** | AI提炼 + 用户判断 |
| **AI介入** | 重 |

| **AI任务** | 1. 解读用户写的信号，推断背后的长期趋势 2. 生成 5-7 个趋势候选（基于用户信号 + 趋势词库补充） 3. 趋势词库（仅作 AI 参考，不直接展示给用户）：AI替代执行岗 / 人机协作常态化 / 注意力成稀缺资源 / 终身职业消失 / 创业自雇比例上升 / 阶层流动收窄 / 多元价值观碰撞 / 全球化与本土化并行 / 社交媒体重塑关系 / 心理问题低龄化 / 屏幕侵蚀身心 / 孤独感蔓延 / 地缘冲突常态化 / 气候与环境危机 / 黑天鹅频率上升 4. 输出 `<!--DATA:{"trendCandidates":["...",...]}--> ` |

**关键**：趋势候选由 AI 根据用户信号动态生成，不是固定的 15 个选项。AI 可以从趋势词库中选，也可以根据用户信号生成新的。

#### N-03 · 素养聚焦 + 价值倾向

| 字段 | 内容 |
|------|------|
| **对应 Word** | 工作纸第四步"素养聚焦" + 第五步"初选与叩问" |
| **目标** | 确定核心素养 + 教育价值取向 |
| **任务类型** | 用户判断 |
| **AI介入** | 无 |
| **展示形式** | 卡片（3 个区域） |
| **用户输入** | ① 最需要的能力：自由填写 1 个（有能力词库提示） ② 最不需要的能力：自由填写 1 个（有能力词库提示） ③ 价值倾向二选一：成功适配 vs 独特创造 |
| **系统输出** | 存储 `{mostNeeded, leastNeeded, valueLeaning}` |
| **AI任务** | 无 |
| **下一步条件** | 三项都填完 → N-04 |

**能力词库**（仅作提示标签，不强制从中选）：
- 最需要参考：复杂问题解决 / 批判性思维 / 创造力 / 同理心 / 韧性 / 自我驱动学习 / 人机协作 / 跨文化沟通

**价值倾向**（工作纸原文）：
- **成功适配**：优先培养孩子适应未来社会、获得主流认可的能力
- **独特创造**：优先保护孩子与众不同的特质，培养他定义和创造新事物的能力

### 模块 W — 我们的根基

> **Word 原意**：连接家族的精神血脉
> **快照产出**：底层代码（生存哲学）/ 源于什么看重 / 继承什么内核 / 升级为什么形式
>
> **⚠️ 设计原则**：本模块像家庭战略咨询师一对一深挖。很多家长自己想不出来，需要 AI 通过多轮追问、举例、引导帮他们"挖"出来。AI 的角色是"帮你看见"，不是"替你选择"。

---

#### W-01 · AI 引导故事回忆（多轮对话）

| 字段           | 内容                                                         |
| -------------- | ------------------------------------------------------------ |
| **对应 Word**  | 工作纸第一步“挖掘情感地标”——回忆让你情绪强烈的瞬间           |
| **目标**       | 通过 2-4 轮咨询式引导，帮助用户从抽象印象进入一个能体现家族行为模式的具体故事 |
| **任务类型**   | 收集信息                                                     |
| **AI介入**     | 重（咨询师式引导）                                           |
| **展示形式**   | AI 多轮对话                                                  |
| **用户输入**   | 每轮自由文本回答                                             |
| **系统输出**   | 存储整轮对话 + 提取的核心故事素材                            |
| **下一步条件** | 已获得足够素材进入 W-02，或达到本 step 的轮次上限后进入结构化确认 |

### 交互原则
- W-01 在 MVP 阶段采用有限轮次引导，而非完全开放式深挖
- 通常 3 轮为主，最多 4 轮
- 每轮只推进一个目标，不堆积多个问题
- AI 的职责是帮助用户回忆、聚焦和初步看见模式，不在此步完成最终判断

### 轮次目标（固定）
**第 1 轮：打开记忆入口**  
帮助用户从“父母常说的话 / 一个强烈瞬间 / 一次冲突或骄傲的时刻”进入回忆。

**第 2 轮：把抽象表述拉到具体事件**  
如果用户回答抽象，AI 要追问到一个具体场景、人物、反应或处理方式。

**第 3 轮：试探背后的价值取舍**  
AI 可基于用户提供的故事，尝试提出一个初步的价值取舍判断，请用户确认或修正。

**第 4 轮：补素材或自然收束（可选）**  
如果素材仍不足，补问一个更具体的瞬间；如果素材已基本足够，则自然收束并进入下一步。

### AI 动态生成原则
- 每一轮的具体问法由 AI 根据用户上一轮回答动态生成
- 不要求逐字复用固定话术
- AI 应优先追问用户已经提到的信息，而不是机械套用通用问题
- 如果用户已经给出具体故事，可跳过部分追问，直接进入下一轮目标
- 如果用户持续抽象，AI 可提供贴近日常生活的提示帮助回忆

### 可参考的引导方式（示例，不是固定脚本）
- 若用户提到一句家里常说的话，可追问这句话背后的具体事件
- 若用户提到一次冲突，可追问当时谁怎么反应、最后怎么处理
- 若用户只给出笼统感受，可请其回忆某次吃饭、做决定、争执、被表扬或被批评的瞬间
- 在价值取舍试探时，AI 可参考“诚信 vs 利益 / 风险 vs 安全 / 规则 vs 人情 / 个人 vs 集体”等框架，但必须结合用户故事动态生成，不得机械罗列

### 罗盘包含（对应 Word）

| Word 步骤 | 内容 | 实现方式 |
|----------|------|---------|
| 第一步：战略输入 | 导入四大快照到罗盘模板 | 自动填充 |
| 第二步：战略推演 | ① 机遇匹配题（S×N）② 矛盾化解题（W vs N）③ 哲学锚定题（E+S+W） | AI 报告生成 |

### 教育愿景模板（Word 原文）

> 我们，【姓名代码】家庭，将运用【家庭优势资本 S】，秉承【升级后的家族精神 W】，在充满【未来关键趋势 N】的时代，坚定选择【价值观方向 E】的道路，全力将孩子培养成一个【结合核心素养与价值基石的、鲜活的人像描述】的人。

### 战略推演三问（AI 报告生成时回答）

1. **机遇匹配**：家庭资本（S）中哪项优势，最适合用来培养未来时代需求（N）中看重的核心素养？→ `___优势 × ___素养`
2. **矛盾化解**：家族精神内核（W）中需要升级的部分，与未来时代需求（N）之间，是否存在紧张或矛盾？如何创造性解决？
3. **哲学锚定**：价值观共识（E）中的战略方向，将如何决定我们利用家庭资本（S）和精神内核（W）的具体方式？

## 五、AI 行为规范

| 规范 | 说明 |
|------|------|
| 每次只问一件事 | 不堆积问题 |
| 不重复卡片内容 | 说"请在卡片中选择"即可，不列出选项 |
| 每段不超过 3 句 | 保持简洁，自然口吻 |
| 不用列表/bullet | 对话感，不是报告感 |
| **AI 不替用户做价值选择** | AI 负责理解、追问、提炼、给 hint，选择权和命名权在用户 |
| **所有选项由 AI 动态生成** | 情绪标签、取舍轴、英雄特质、参考命名、副作用猜测、升级方向——全部基于用户上下文生成 |
| 快照 = 判断句沉淀 | 不是文案润色，必须对应 Word 模板的每个空 |
| W 模块像咨询师 | 多轮追问、举例、引导，帮用户"挖"出来，不急着收结论 |
| 结构化数据输出 | W-03 AI 输出 `<!--DATA:{...}-->`（从故事中提取取舍轴）；S-05 N-04 W-07 E-05 输出 `<!--SNAPSHOT:...-->` |
