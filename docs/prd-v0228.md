## 一、产品概述

### 1.1 产品定位

一款配合彼灯教育「家庭教育战略 CEO 项目」线下工作坊使用的 AI 辅助对话工具。通过 AI 对话 + 嵌入式交互卡片的形式，引导家长完成家庭教育愿景定位，最终生成一份完整的《家庭定位罗盘》和《教育战略愿景宣言》。

### 1.2 核心价值主张

- **对家长**：从纸质工作纸升级为智能对话体验，AI 帮助深化思考、优化表达，20-30 分钟内完成愿景定位
- **对课程**：配合线下教学节奏，老师负责讲解和引导，产品负责记录和深化，各司其职
- **对业务**：课程的电子化延伸，提升课程专业感和交付质量，增强用户粘性

### 1.3 成功指标

- 完成率：>80% 的用户在一次 session 中完成全部 5 个模块
- 时长：平均完成时间 20-30 分钟
- 满意度：用户对最终愿景宣言的认可度（确认时不修改/仅微调 > 60%）

## 二、用户画像与使用场景

### 2.1 核心用户

参加彼灯教育线下工作坊的家长（夫妻双方），主要特征：
- 关注子女教育，有战略思考意愿
- 非技术背景，需要简洁直观的交互
- 在线下课堂环境中使用（笔记本电脑或手机）

### 2.2 主要使用场景

**场景 A：课堂同步使用（主场景）**
家长在线下工作坊中，老师讲完一个模块后，打开工具完成对应模块的记录和深化。老师控制节奏，工具配合推进。

**场景 B：课后独立补充**
课堂上未完成的部分，家长回家后独立完成或与伴侣讨论后补充修改。

**场景 C：回顾与修改**
家长随时回到工具中查看/修改已完成的快照和愿景宣言。

## 三、产品架构

### 3.1 整体交互模式

**全屏对话为主体 + 嵌入式交互卡片 + 顶部进度导航**

```
┌─────────────────────────────────────────────┐
│  [家底] ─── [眼光] ─── [根基] ─── [共识] ─── [罗盘]  │  ← 顶部进度条
│  ● 完成    ● 完成    ◉ 进行中   ○ 未开始  ○ 未开始   │
├─────────────────────────────────────────────┤
│                                             │
│  AI 对话消息                                  │  ← 主体区域：
│  AI 对话消息                                  │     全屏对话流
│  ┌───────────────────────┐                  │
│  │   嵌入式交互卡片        │                  │  ← 卡片嵌入在
│  │   （选择/填写/快照）     │                  │     对话流中
│  └───────────────────────┘                  │
│  用户回复消息                                  │
│  AI 对话消息                                  │
│                                             │
├─────────────────────────────────────────────┤
│  [输入框：在这里回复...]            [发送]     │  ← 底部输入区
└─────────────────────────────────────────────┘
```

### 3.2 设计原则

1. **对话是主线，卡片是节点**：用户感受是"在跟AI顾问聊天"，不是"在填表"
2. **两张卡片之间至少一轮AI对话**：防止退化为表单体验
3. **AI 追问不超过2轮**：保持节奏紧凑，每个模块控制在 5-7 分钟
4. **快照是每个模块的仪式感收尾**：自动生成 + 用户可编辑
5. **进度条始终可见**：用户随时知道自己在旅程的哪一站

## 四、模块流程详细设计

### 4.1 总体流程

```
模块S（家底）→ 模块N（眼光）→ 模块W（根基）→ 模块E（共识）→ 模块R（罗盘整合）
```

每个模块内部遵循统一节奏：
```
AI 引导开场 → 交互卡片（用户记录）→ AI 追问深化（1-2轮）
→ 交互卡片（用户记录）→ AI 汇总 → 快照预览卡（用户确认）
```

---

### 4.2 模块 S：我们的家底

**目标**：看清家庭资源坐标，明确优先升级方向
**产出**：《家底快照》
**预估时长**：5-6 分钟

#### 对话流

| 序号 | 类型 | 内容 |
|------|------|------|
| S-01 | 🤖 AI | 开场引导：介绍"家底盘点"的目的，解释三大资本的概念（经济/文化/社会）。如在课堂场景，可简化为"老师刚刚讲的三大资本，现在我们来做一个定位。" |
| S-02 | 🎴 卡片 | **资本矩阵卡（C6）**：三行表格，每行选等级 + 写关键词 |
| S-03 | 🤖 AI | 接住用户选择，做一段分析。重点关注：三项资本之间的关系、最高/最低项的特征、用户可能没意识到的模式。追问1轮（如"你们的文化资本优势有没有在撬动其他两项？"） |
| S-04 | 👤 用户 | 打字回复 AI 的追问 |
| S-05 | 🤖 AI | 接住回复，引出战略决策 |
| S-06 | 🎴 卡片 | **单选卡（C1）**：最需要优先升级的资本是？（经济/文化/社会） |
| S-07 | 🤖 AI | 基于选择做解读（如选了高项→"选择把长板拉更长"；选了低项→"选择补短板"），然后汇总生成快照 |
| S-08 | 🎴 卡片 | **快照预览卡（C7）**：展示《家底快照》，用户可编辑/确认 |

#### AI 对话要点
- 分析三项资本的高低组合模式（如"文化高+社会低 = 内功强但连接少"）
- 追问时关注资本之间的互相撬动关系
- 不评判等级高低，强调"这是资源地图，不是成绩单"

### 4.3 模块 N：我们的眼光

**目标**：锚定对未来趋势的判断，押注核心素养
**产出**：《眼光快照》
**预估时长**：4-5 分钟

#### 对话流

| 序号 | 类型 | 内容 |
|------|------|------|
| N-01 | 🤖 AI | 开场引导：从"家底"过渡到"眼光"，引导用户思考未来趋势。如在课堂，衔接老师刚讲的趋势内容 |
| N-02 | 🎴 卡片 | **标签选择卡（C2）单选**：你认为最核心的未来趋势是？词库：AI替代执行岗 / 全球竞争合作 / 心理健康重要性上升 / 创造力成为硬通货 / 终身学习常态化 / 虚拟与现实融合 / + 自定义 |
| N-03 | 🤖 AI | 接住选择，做简要分析，引出素养选择 |
| N-04 | 🎴 卡片 | **标签选择卡（C2）单选**：面对这个趋势，孩子最需要的能力是？词库：复杂问题解决 / 批判性思维 / 创造力 / 同理心 / 韧性 / 自我驱动学习 / 人机协作 / 跨文化沟通 / + 自定义 |
| N-05 | 🤖 AI | 连接家底快照做交叉分析（如"你们选了创造力，而你们的文化资本偏向知识对话——这意味着你们家庭已经有了培养创造力的土壤"），追问1轮深化 |
| N-06 | 👤 用户 | 打字回复 |
| N-07 | 🤖 AI | 汇总，生成快照 |
| N-08 | 🎴 卡片 | **快照预览卡（C7）**：展示《眼光快照》，用户可编辑/确认 |

#### AI 对话要点
- 关键动作：**跨模块连接**——把眼光的选择和家底的数据交叉分析
- 帮用户看到趋势判断和自身资源之间的关系
- 不灌输"正确"趋势，尊重用户的判断

### 4.4 模块 W：我们的根基

**目标**：发现家族精神内核，完成继承与升级的选择
**产出**：《根基快照》
**预估时长**：6-7 分钟（四个模块中最长，因为涉及故事和反思）

#### 对话流

| 序号 | 类型 | 内容 |
|------|------|------|
| W-01 | 🤖 AI | 开场引导：解释"家族精神内核"的概念——不是美德清单，而是代代相传的生存哲学。提供几组对比例子帮助打开记忆（诚信vs利益、风险vs安全、个人vs集体、规则vs人情） |
| W-02 | 🎴 卡片 | **短文填写卡（C4）**："描述一个让你感受到'我们家就会这么办'的瞬间" |
| W-03 | 🤖 AI | **这是AI价值最高的环节**。基于用户的故事，帮助提炼和命名。如"你描述的这个瞬间里，爷爷宁可吃亏也要守信——如果我把你们家的精神内核概括为'宁折不弯的信义哲学'，你觉得准确吗？" 追问1轮 |
| W-04 | 👤 用户 | 打字回复，确认/修正 AI 的命名 |
| W-05 | 🤖 AI | 引导"战略诊断"——这套内核在当下是赋能还是约束？ |
| W-06 | 🎴 卡片 | **关键词填写卡（C3）复合**：我们决心继承的核心精神是「___」，升级为适应未来的「___」 |
| W-07 | 🤖 AI | 追问升级宣言的可操作性（如"你说从'必须考第一'升级为'在热爱领域做到极致'，下次孩子考了第三名回来，你们会怎么回应？"） |
| W-08 | 👤 用户 | 打字回复 |
| W-09 | 🤖 AI | 汇总，生成快照 |
| W-10 | 🎴 卡片 | **快照预览卡（C7）**：展示《根基快照》，用户可编辑/确认 |

#### AI 对话要点
- **命名能力是核心**：很多家长知道自己家族有某种精神，但说不出来。AI 帮他们找到精准的语言
- 语气温暖、不评判："这不是对错的问题，而是时代变了，表现形式需要更新"
- 追问升级路径时要**具体到场景**，不要停在口号层面

### 4.5 模块 E：我们的共识

**目标**：从24个教育价值观中锚定3个战略基石，确定方向
**产出**：《共识快照》
**预估时长**：5-6 分钟

#### 对话流

| 序号 | 类型 | 内容 |
|------|------|------|
| E-01 | 🤖 AI | 开场引导：在看到价值观选项之前，先做直觉锚定 |
| E-02 | 🎴 卡片 | **关键词填写卡（C3）**：两个填空——"孩子最不能缺少的品格是___"；"我最担忧孩子缺少___" |
| E-03 | 🤖 AI | 接住直觉，简短回应，然后引出价值观画廊 |
| E-04 | 🎴 卡片 | **价值观画廊卡（C8）**：从24个价值观中选出3个核心 + 2个暂缓。24个价值观分6大类：内在稳定（真实一致/情绪安全/自尊自信/韧性复原）、成长动力（自主驱动/成长思维/好奇探索/创造创新）、关系连接（深度同理/有效沟通/边界清晰/协作共赢）、精神意义（意义追寻/审美感知/开放包容/内在和解）、社会参与（责任担当/批判思维/公平正义/社会贡献）、实用维度（学业成就/财务智慧/实用技能/社会声誉） |
| E-05 | 🤖 AI | 分析用户的选择模式（如"你的3个核心都来自'成长动力'和'精神意义'两个大类，这说明你们更看重孩子的内在世界"）。对比直觉锚定和最终选择的一致性/差异。追问1轮 |
| E-06 | 👤 用户 | 打字回复 |
| E-07 | 🎴 卡片 | **单选卡（C1）三选一**：你们的战略方向更偏向？内核（向内探索、夯实自我）/ 创造（向外开拓、定义新事物）/ 连接（搭建关系、成就他人） |
| E-08 | 🤖 AI | 汇总，生成快照 |
| E-09 | 🎴 卡片 | **快照预览卡（C7）**：展示《共识快照》，用户可编辑/确认 |

#### AI 对话要点
- 分析选择的**大类分布模式**，帮用户看到自己的价值观倾向
- 关注直觉锚定和最终选择的**一致性/冲突**——冲突点往往是最有洞察价值的
- 暂缓项同样重要——"选择不做什么"本身就是战略决策

### 4.6 模块 R：罗盘整合

**目标**：将四张快照交叉整合，生成最终的教育战略愿景宣言
**产出**：《教育战略愿景宣言》+《战略承诺书》
**预估时长**：4-5 分钟

#### 对话流

| 序号 | 类型 | 内容 |
|------|------|------|
| R-01 | 🤖 AI | 展示四张快照的汇总，引导用户进入"战略推演"。"现在让我们把前面四个维度放在一起看——" |
| R-02 | 🤖 AI | **机遇匹配**：AI 基于家底（S）和眼光（N）的数据，提出"你们的___资本优势，最适合用来培养___素养"的推演，让用户确认或调整 |
| R-03 | 👤 用户 | 回复确认/调整 |
| R-04 | 🤖 AI | **传承校准**：AI 基于根基（W）和共识（E）的数据，分析家族精神升级版与价值观基石的呼应关系 |
| R-05 | 👤 用户 | 回复确认/调整 |
| R-06 | 🤖 AI | 整合所有数据，生成愿景宣言初稿 |
| R-07 | 🎴 卡片 | **快照预览卡（C7）加强版**：展示《教育战略愿景宣言》。模板：「我们，【姓名代码】家庭，将运用【家庭优势资本 S】，秉承【升级后的家族精神 W】，在充满【未来关键趋势 N】的时代，坚定选择【价值观方向 E】的道路，全力将孩子培养成一个【结合核心素养与价值基石的人像描述】的人。」用户可深度编辑 |
| R-08 | 👤 用户 | 确认愿景宣言 |
| R-09 | 🤖 AI | 仪式感收尾，展示承诺书 |

#### AI 对话要点
- 这是 **AI 价值最高的环节**——前四个模块各自独立，这里做交叉连接
- AI 要帮用户发现四个象限之间的"战略交点"
- 愿景宣言初稿要**有温度**，不能是干巴巴的填词，要像一段有感情的宣言

## 五、嵌入式交互卡片规格

### 5.1 卡片类型总表

| 编号 | 名称 | 交互方式 | 使用次数 |
|------|------|----------|---------|
| C1 | 等级/单选卡 | 从3-4个预设选项中选1个 | 3次 |
| C2 | 标签选择卡 | 从词库中选1个或多个标签，支持自定义 | 2次 |
| C3 | 关键词填写卡 | 写1-3个短关键词 | 2次 |
| C4 | 短文填写卡 | 写1-3句话 | 1次 |
| C6 | 资本矩阵卡 | 三行复合卡片：每行选等级+写关键词 | 1次 |
| C7 | 快照预览卡 | 展示AI生成文字，可编辑/重新生成/确认 | 5次 |
| C8 | 价值观画廊卡 | 从24项中选5项（3核心+2暂缓） | 1次 |

### 5.2 卡片通用规范

- 所有卡片嵌入在对话消息流中，视觉上与普通消息有区别（如有卡片边框/背景色）
- 每张卡片有明确的「确认」按钮，点击后卡片状态变为"已完成"（不可再改，需点"编辑"才能修改）
- 已完成的卡片在对话流中保持可见，但视觉上变为收起/灰化状态
- 所有卡片在移动端和桌面端均需可用

### 5.3 关键卡片详细规格

#### C6 资本矩阵卡

```
┌─────────────────────────────────────────────────┐
│  请为你们的家庭做一个资本共识定位：                   │
│                                                 │
│  经济资本      ○ L1  ● L2  ○ L3    [关键词___]  │
│  "燃料与安全垫"                                   │
│                                                 │
│  文化资本      ○ L1  ○ L2  ● L3    [关键词___]  │
│  "氛围与知识库"                                   │
│                                                 │
│  社会资本      ● L1  ○ L2  ○ L3    [关键词___]  │
│  "网络与连接器"                                   │
│                                                 │
│                                  [确认 →]       │
└─────────────────────────────────────────────────┘
```
- 三行可独立操作，不要求按顺序填
- 关键词为可选填写（有 placeholder 示例）
- 等级选择为必填

#### C7 快照预览卡

```
┌─────────────────────────────────────────────────┐
│  📸 你们的XX快照                                  │
│  ───────────────────────                        │
│                                                 │
│  [可编辑的富文本区域]                              │
│  AI 基于用户所有输入自动生成的段落                   │
│  用户可直接点击修改任何文字                          │
│                                                 │
│  [🔄 重新生成]           [✅ 确认保存]             │
└─────────────────────────────────────────────────┘
```
- "重新生成"：AI 用不同的表达方式重写快照（保持数据不变，换措辞）
- "确认保存"：锁定快照，进入下一模块
- 确认后仍可通过进度条回溯修改

│                                                 │
│  ┌─内在稳定─┐ ┌─成长动力─┐ ┌─关系连接─┐          │
│  │真实一致  │ │自主驱动  │ │深度同理  │          │
│  │情绪安全  │ │成长思维  │ │有效沟通  │          │
│  │自尊自信  │ │🟠好奇探索│ │边界清晰  │          │
│  │韧性复原  │ │🟠创造创新│ │协作共赢  │          │
│  └─────────┘ └─────────┘ └─────────┘          │
│  ┌─精神意义─┐ ┌─社会参与─┐ ┌─实用维度─┐          │

## 六、AI 对话设计规范

### 6.1 AI 人设

- 角色：家庭教育战略顾问，温暖、专业、不说教
- 语气：像一个有经验的朋友，而不是老师或权威
- 语言：中文，口语化但不随意，避免学术腔

### 6.2 AI 能力边界

**AI 应该做的**：
- 帮用户命名（把模糊的感觉变成精准的语言）
- 跨模块连接（把A模块的数据和B模块的选择关联起来分析）
- 温和追问（帮用户看到自己没注意的模式或盲区）
- 生成和优化文字（快照、愿景宣言）

**AI 不应该做的**：
- 评判用户的选择（没有"对"或"错"的价值观）
- 过度追问（每个节点最多1-2轮，不要变成心理咨询）
- 给出教育建议（工具是帮用户想清楚自己的愿景，不是给方案）
- 替用户做决定（所有关键选择必须由用户确认）

### 6.3 AI 追问策略

每个模块中，AI 的追问从以下策略池中选择**最多1个**执行：

| 策略 | 描述 | 适用模块 |
|------|------|---------|
| 关联分析 | 连接当前模块与前一模块的数据 | N、W、E、R |
| 命名辅助 | 帮用户把抽象感受提炼为精准语言 | W |
| 场景具象化 | 追问"在具体场景里这意味着什么" | W、E |
| 一致性检验 | 对比用户不同节点的选择是否一致 | E、R |
| 盲区提示 | 温和指出用户可能忽略的维度 | S、N |

### 6.4 数据传递

AI 在各模块间需要传递和引用的数据：

```
模块S → 三项资本等级+关键词、优先升级选择
         ↓ 传递给 N（用于交叉分析）、R（用于罗盘整合）

模块N → 核心趋势、押注素养
         ↓ 传递给 R（用于罗盘整合）

模块W → 关键瞬间、精神内核命名、继承×升级
         ↓ 传递给 R（用于罗盘整合）

模块E → 直觉锚定、3基石+2暂缓、战略方向
         ↓ 传递给 R（用于罗盘整合）

模块R → 汇总所有数据 → 生成愿景宣言
```

## 十、待定与后续迭代

### 10.1 本版本暂不包含

- 夫妻双人协作模式（双方在同一 session 中各自填写再合并）
- 年度复盘提醒功能
- 愿景宣言的精美排版/海报导出
- 与其他课程模块（非愿景定位）的连接

### 10.2 需进一步讨论的问题

1. **课堂同步机制**：老师是否需要一个"控制台"来统一推进全班的模块节奏？
2. **夫妻协作**：当前设计是一份设备上夫妻共同完成。是否需要支持各自设备独立填写后合并？
3. **快照的可视化**：是否需要把四张快照和罗盘做成一张可视化的图（而不仅仅是文字）？
4. **24个价值观的顺序和分组**：是否需要根据用户前面模块的回答动态排序？
